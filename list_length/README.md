# ListLength

## Desafio 1 - Ignite Elixir

Criar uma função que, dada uma lista de números (exemplo [1, 3, 6, 43, 6]), calcule, recursivamente, quantos elementos há na lista e retorne esse valor.

```bash
iex> ListLength.call([1, 2, 3, 5, 7])
5
```
